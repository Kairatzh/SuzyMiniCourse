## Обзор проекта

### Что такое Fill AI?

**Fill AI** — это онлайн-платформа для автоматического создания мини-курсов с использованием ИИ-агентов. Система генерирует:

-  Конспекты по теме
-  Тестовые задания
-  Подборку видео с YouTube
-  Граф знаний для навигации

### Основные возможности

-  Генерация курсов через ИИ
-  Хранение курсов в базе данных
-  Граф знаний для структурирования курсов
-  Telegram бот для быстрого доступа
-  RESTful API для интеграции

### Целевая аудитория

-  Подростки и студенты
-  Люди, изучающие новое
-  Преподаватели
-  Образовательные организации

## Компоненты проекта

### 1. AI Service (`ai_service/`)

**Назначение:** Генерация курсов с использованием ИИ-агентов.

**Основные модули:**
- `agents/` — ИИ-агенты для генерации
- `tools/` — инструменты агентов (summary, tests, videos)
- `llm/` — интеграция с LLM (OpenRouter, OpenAI, Together)
- `prompt_engineering/` — промпты для агентов
- `api/` — REST API для генерации

**Endpoints:**
- `POST /api/v1/generate_main/generate` — генерация курса

### 2. Backend Service (`backend_service/`)

**Назначение:** Основной бэкенд для управления курсами и пользователями.

**Основные модули:**
- `api/` — REST API endpoints
- `database/` — подключения к БД (PostgreSQL, Neo4j)
- `models/` — модели данных (SQLAlchemy)
- `schemas/` — Pydantic схемы
- `services/` — бизнес-логика
- `auth/` — аутентификация (JWT)

**Endpoints:**
- `POST /api/v1/users/register` — регистрация
- `POST /api/v1/users/login` — вход
- `POST /api/v1/courses/generate` — генерация курса
- `GET /api/v1/courses/{id}` — получение курса
- `GET /api/v1/courses/graph` — граф знаний

### 3. Frontend Service (`frontend_service/`)

**Назначение:** Веб-интерфейс для пользователей (в разработке).

**Технологии:**
- React
- TailwindCSS
- React Router

### 4. Telegram Bot (`backend_service/src/bot.py`)

**Назначение:** Telegram бот для быстрого доступа к генерации курсов.

## Быстрый старт

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   cd backend_service && pip install -r requirements.txt
   ```

2. **Настройте базы данных:**
   - PostgreSQL
   - Neo4j

3. **Создайте `.env` файлы:**
   - См. `.env.example` в каждом сервисе

4. **Запустите сервисы:**
   ```bash
   # AI Service
   cd ai_service && uvicorn api.main:app --port 8000
   
   # Backend Service
   cd backend_service && python -m src.main
   ```
